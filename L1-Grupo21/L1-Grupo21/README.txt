Se hizo en dos carpetas distintas, la carpeta Cliente que contiene cliente.go y la carpeta Servidor que contiene servidor.go.
Se debe dividir la terminal y hacer correr primero el servidor con el comando: 
	go run servidor.go 
Luego el cliente con el comando
	go run cliente.go
Es muy importante que se ejecute primero el servidor y luego el cliente
Con esto funciona el programa de manera optima y eficaz.

Se va alternando entre el cliente y el servidor pidiendole a ambos que letra atacar en el mapa
El cliente se marca como 1 en el mapa y el servidor como 2 en el mapa

Igualmente se tiene en consideración que solo serán inputs válidos
